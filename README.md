# Smart Hospital Management System

## Project Overview

The Smart Hospital Management System is a software engineering project designed to improve the management of hospital operations through digital technology. The system helps manage patient records, appointment scheduling, and communication between healthcare providers and patients.

## Sustainable Development Goal (SDG)

This project supports **SDG 3: Good Health and Well-being** by promoting efficient healthcare service delivery and improving access to healthcare information.

## Objectives

* Improve hospital record management.
* Facilitate appointment scheduling.
* Enhance communication between patients and healthcare providers.
* Reduce paperwork and administrative delays.

## System Features

* User Registration and Login
* Patient Record Management
* Appointment Scheduling
* Doctor and Patient Dashboard
* Report Generation
* User Management

## Technologies Used
